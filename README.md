# GitHub Actions 在线云编译路由固件


## 说明

-  `*.yml` 自动编译工作流
-  `Plug-in` 插件目录
-  `*.config` 路由型号配置单
-  `Part1.sh` 自定义软件仓库
-  `Part2.sh` 个性化配置
